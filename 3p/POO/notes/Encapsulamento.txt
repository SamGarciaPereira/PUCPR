Atributo (características) = privado
Método (comportamentos) = público

Exemplo:

public class Pessoa{
    private String nome;
    private double peso;
}

public void falar(){
}

public void andar(){
}

Definição: consiste em separar os aspectos externos de um objeto, que são
acessíveis para outros objetos, dos detalhes internos de implementação do objeto.
