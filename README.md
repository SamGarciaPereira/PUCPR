# 🎓 Computer Science @ PUCPR - Coursework Archive

Welcome to my academic repository! This space is dedicated to storing the projects, exercises, and lab assignments developed during my **Computer Science** degree at **Pontifícia Universidade Católica do Paraná (PUCPR)**. 

The goal of this repository is to track my progress, organize my study materials, and build an accessible portfolio of my academic journey.

---

## 📂 Repository Structure

The folders are organized by semesters and correspond to the official curriculum subjects. 

### 📁 `1p/` (1st Semester)
* **`ExperienciaCriativaNavegandoNaComputacao/`**: Creative Experience: Navigating in Computing.
* **`RaciocinioAlgoritmico/`**: Algorithmic Reasoning (Logic and Python).
* **`ResolucaoDeProblemasComLogicaMat/`**: Problem Solving with Mathematical Logic (C/C++ focus).

### 📁 `2p/` (2nd Semester)
* **`PerformanceEmSistemasCiberfisicos/`**: Performance in Cyber-Physical Systems (Includes Smart Cabinet project).
* **`ProgramacaoImperativa/`**: Imperative Programming.
* **`ProgramacaoWeb/`**: Web Programming fundamentals.

### 📁 `3p/` (3rd Semester)
* **`ExperienciaCriativaCriandoSolucoesComputacionais/`**: Creative Experience: Creating Computational Solutions.
* **`POO/`**: Object-Oriented Programming (Java).
* **`SegurancaDaInformacao/`**: Information Security.

### 📁 `4p/` (4th Semester)
* **`BigData/`**: Big Data fundamentals and analysis.
* **`ResolucaoProblemasEstruturadosComputacao/`**: Structured Problem Solving in Computing (Data Structures like Stacks, Queues).
* **`SistemasOperacionaisCiberfisicos/`**: Cyber-Physical Operating Systems (Embedded systems and FreeRTOS).
